# Ruxsatlar (Permissions)

Linuxda fayl ruxsatlari quyidagicha belgilanadi:

| Raqam | Ma'nosi |
| :--- | :--- |
| 7 | O'qish, Yozish, Ishga tushirish (rwx) |
| 5 | O'qish va Ishga tushirish (r-x) |
| 4 | Faqat o'qish (r--) |

Misol: `chmod 777 script.sh`
